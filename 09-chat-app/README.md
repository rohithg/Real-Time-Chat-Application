# Real-Time Chat Application

Full-featured chat application with Socket.io for real-time messaging, rooms, and user presence.

## Features

- **Real-time Messaging**: Instant message delivery with Socket.io
- **Multiple Rooms**: Switch between different chat rooms
- **User Presence**: Live user list with online status
- **Typing Indicators**: See when others are typing
- **Message History**: Persistent chat history
- **Private Messaging**: Direct messages between users
- **System Messages**: Join/leave notifications

## Technologies

- Socket.io (WebSocket)
- Express.js
- Real-time event handling
- Vanilla JavaScript

## Installation

```bash
npm install
npm start
```

Visit http://localhost:3003

## Skills Demonstrated

1. **WebSocket Communication**: Bidirectional real-time data
2. **Event-Driven Architecture**: Socket.io events
3. **State Management**: User and room tracking
4. **UI Updates**: Dynamic DOM manipulation
5. **Presence Detection**: Online user tracking
6. **Message Broadcasting**: Room-based messaging
